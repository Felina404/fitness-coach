# Fitness coach Landing Page

## Live Demo
[https://fitness-coach-landing.netlify.app/](https://fitness-coach-landing.netlify.app/)

## Features
- Fully responsive
- smooth Scrolling
- Animations

## Tech Stack
- Vite
- React
- Tailwind CSS
- TS

## Installation
- git clone https://github.com/Felina404/fitness-coach.git 
- cd fitness-coach
- npm install
- npm run dev

### build
- npm run build
### Preview
- npm run preview
